A Jekyll Blog About Jekyll

We like Jekyll and you can, too! This site was created to serve as a resource for anyone who wants to learn Jekyll. There are lots of great ways to create blogs and websites, and Jekyll is one of them. We're big fans of Jekyll because it is:

  • Simple
  • Blog-aware
  • Static

We like using Jekyll because, in 90% of cases, it's a better solution than a CMS (content management system). With Jekyll, you don't have to deal with PHP, databases, and most of the security, scalability, and maintenance challenges associated with them. However, Jekyll can be challenging for new users who aren't comfortable with the command line, Markdown, YAML, version control systems like Git, and the Liquid templating language. The articles on this site are intended to help more people overcome those initial challenges so they can share in the joyous experience that is Jekyll.

If you're new here, we recommend you start with the article Your First 15 Minutes With Jekyll (coming soon).

Or, if you like videos, check out this tutorial: Create Static Sites with Jekyll And GitHub Pages
